# 实验说明

## 研究问题

外部证据、事实验证和保守拒答能否降低大语言模型事实问答中的幻觉？

## 对比方法

1. Base LLM：本地 Qwen2 0.5B 直接回答。
2. RAG：检索最高相关文档后抽取证据。
3. RAG + Verify：相关性不足或候选差距过小时拒答。
4. RAG + CoVe：在检索基础上增加关键词独立核查。

## 数据

- 100 条事实性问题。
- 101 个知识文档，包括直接证据与近似干扰材料。
- 类别涵盖常识、操作系统、软件工程和数据库。

## 关键修复

### 评测器

原评测器要求整句精确匹配，造成大量假阳性。修复后采用：

- Unicode 归一化；
- 字符 n-gram 召回；
- 关键事实覆盖；
- 数值一致性检查；
- 拒答与调用失败单独统计。

### 检索器

原离线降级方案使用随机哈希向量，没有语义检索能力。修复后使用中文字符 n-gram TF-IDF，保证无需下载模型也可稳定复现。

### 工程流程

- DeepSeek 不可用时自动切换 Ollama。
- 每道题写入 checkpoint。
- 保存检索上下文和逐题判定。
- 提供 `--evaluate-only` 离线复评模式。

## 结果解读

RAG 和验证机制降低了幻觉率，但 Verify 与 CoVe 的覆盖率同步下降。尤其是 CoVe，幻觉率最低但覆盖率仅 14%，说明高风险系统需要在“少犯错”和“能够回答”之间选择合适阈值。
